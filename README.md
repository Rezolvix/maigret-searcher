# Maigret Username Searcher

Простая консольная программа на Python для поиска аккаунтов по имени пользователя на различных веб-сайтах с использованием библиотеки [Maigret](https://github.com/soxoj/maigret).

## Возможности
- Поиск по имени пользователя на сотнях популярных сайтов.
- Использование асинхронного API для быстрого сканирования.
- Извлечение дополнительной информации из профилей (если доступно).
- Настраиваемый лимит сайтов и таймаут.

## Требования
- Python 3.10 или выше.
- Библиотека `maigret`.

## Установка

1. Склонируйте репозиторий или скачайте файл `maigret_searcher.py`.
2. Установите необходимые зависимости:

```bash
pip install maigret
```

## Использование

Запустите скрипт из терминала:

```bash
python maigret_searcher.py
```

После запуска программа перейдет в интерактивный режим, где предложит:
1. Ввести **имя пользователя** для поиска.
2. Указать **количество сайтов** для проверки (по умолчанию 500).
3. Указать **таймаут** запросов (по умолчанию 30 секунд).

Для выхода из программы введите `exit` или `выход` в поле ввода имени пользователя.

## Особенности работы
- **Чистый вывод**: Программа автоматически подавляет технические ошибки (например, ошибки активации Vimeo или 403 Forbidden), чтобы вы видели только прогресс и результат.
- **Обработка ошибок**: Если сайт защищен Cloudflare или требует капчу, программа просто пропустит его, не прерывая общий поиск.
- **Поддержка Windows**: Исправлена проблема с отображением русских символов в консоли.

## О проекте
Эта программа является оберткой над библиотекой Maigret, которая позволяет эффективно проводить OSINT-исследования по именам пользователей.
